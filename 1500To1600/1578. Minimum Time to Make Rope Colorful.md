# 1578. Minimum Time to Make Rope Colorful

題目網址 : [1578. Minimum Time to Make Rope Colorful](https://leetcode.com/problems/minimum-time-to-make-rope-colorful/description)

## 題目描述

Alice has `n` balloons arranged on a rope. You are given a **0-indexed** string `colors` where `colors[i]` is the color of the `ith` balloon.

Alice wants the rope to be **colorful**. She does not want **two consecutive balloons** to be of the same color, so she asks Bob for help. Bob can remove some balloons from the rope to make it **colorful**. You are given a **0-indexed** integer array `neededTime` where `neededTime[i]` is the time (in seconds) that Bob needs to remove the `ith` balloon from the rope.

Return _the **minimum time** Bob needs to make the rope **colorful**_.

## 限制條件

- `n == colors.length == neededTime.length`
- `1 <= n <= 105`
- `1 <= neededTime[i] <= 104`
- `colors` contains only lowercase English letters.

## 解題思路

根據題目描述，為了要讓汽球達成`colorful`的條件，必需移除相連汽球中多出的部分
且每移除一個汽球，會花費不同的代價，題目要求要讓這個代價為最小
可以馬上想到的是，從最小代價的汽球開始移除
但既然只會留下一個汽球，那便可以反過來思考，選擇最大代價的汽球留下其它的全移除
所以這題累計了相同顏色且連續的汽球的代價並減去最大代價的汽球

## Code

C++版本

```C++
class Solution {
public:
    int minCost(string colors, vector<int> &neededTime) {
        int max = neededTime[0];
        int cost = 0;
        long long sum_of_cost = neededTime[0];
        char prev_color = colors[0];
        for (int i = 1; i <= colors.size(); ++i) {
            if (i == colors.size() || colors[i] != prev_color) {
                cost += (sum_of_cost - max);
                if (i < colors.size()) {
                    prev_color = colors[i];
                    sum_of_cost = neededTime[i];
                    max = neededTime[i];
                }
            } else {
                sum_of_cost += neededTime[i];
            }
            if(i < colors.size() && neededTime[i] > max)
                max = neededTime[i];
        }
        return cost;
    }
};
```
